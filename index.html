1.Surgimento da Web

Os primeiros computadores surgem com a função de realizar cálculos complexos, mas sua função avançou drasticamente com o advento de novas tecnologias e ideias. Posterior a esse período o britânico Tim Berners-Lee aproveitou-se de toda a evolução tecnológica da época e crio a World Wide Web conhecido atualmente pela sigla "www". O britânico viu a possibilidade de criar hipertextos para permitir que várias pessoas trabalhassem juntas, acessando os mesmos documentos. Os hipertextos foram denominados como HTML significando Hiper Text Markup Language, traduzindo para o português: Linguagem de Marcação de Hipertexto. O HTML, atualmente é um componente básico na web, permitindo desenvolver a estrutura básica de um website.

2.Clients X Servers: O que são?

Clients: Dispositivos que consomem a informação provinda dos servidores.

Servers: Dispositivos que armazenam a informação a ser provida aos clientes quando solicitado.

Observação: A relação entre cliente e servidor, pode ser direta, não havendo conexão via internet. Sendo apenas entre cliente "dispositivo" e servidor.

3.Linguagens de Programação

Conceito: Conjunto de informações que são inseridas no dispositivo e o mesmo executa mediante à aquela lógica implementada.

HTML: O HTML não se classifica como uma linguagem de programação, apesar de o mesmo conter o termo "linguaguem" em seu acrônimo o mesmo é um arquivo de marcação de texto.

2.Tags

Conceito: Estruturas de linguagem de marcação contendo instruções a serem renderizadas na página.

Atributos: Propriedades que uma tag pode ter, podendo ser genérica, ou seja, serve a qualquer tag, ou especiais destinada a tags específicas.

Apartir das tags, é possível desenvolver uma gama de marcações dentro do HTML, variando entre textos, listas, tamanho da fonte e até mesmo a cor da marcação.
